# Puntos para mejorar software
* Inventario.
* Ventas.
* Ingreso de facturas: con número de contrato de la factura y se carga las compras que se ha comprado desde un proveedor. 
* Cruce de ventas y compras para sacar la utilidad.
* Ventas diarias.
* Facturas electrónicas: número de cédula e la persona(natural) or empresa(RUC), le envía al SRI, a la persona y a nosotros. 
* Transferencia de dinero para pago con tarjeta debito.
* Ingresar inventario en bloqueo
